# RackMaster Boilerplate

This workspace contains a Django REST backend and a React frontend (Material UI) with Docker setup.

Services:
- backend: Django REST API on port 8000
- frontend: React app on port 3000

Quick start (Docker required):

```bash
docker-compose up --build
```

Backend API sample endpoint: http://localhost:8000/api/hello/
Frontend app: http://localhost:3000
